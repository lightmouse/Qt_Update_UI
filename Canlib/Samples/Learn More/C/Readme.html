C samples

This is a collection of C programs which all demonstrate various features of Canlib.

Importing Canlib

Many of these samples are in the form of Visual Studio projects. These need to have a link to the Canlib headers. By default, they have a relative link from their directory in the installation path, but if you choose to move the samples somewhere else, you need to add a new link. This is done under Project Properties, C/C++, General, Additional Include Directories.

candb

An example of a program that can create, modify and use databases. When executed, it creates a small example database called temp.dlc.

candump

A program which dumps any incoming messages to the screen. It is a bit like the one in Part 2 of the tutorial but with more options. For usage instructions, run candump.exe -?

channeldata

Displays information about all the CAN channels. It does this using calls to canGetChannelData.

echo

Listens for messages on a channel. When it receives a message, it increases its identifier by 1 and sends it back.

gensig

Generates various signals and sends them as CAN messages. These can later be read by a program like the one in Part 6 of the tutorial.

kvrConfig

Demonstrates how to configure remote devices using Kvrlib. A virtually identical program in C# with a more in-depth readme is available.

kvrConnect

Shows how to discover devices using Kvrlib. This program also has a C# equivalent with more information.

kvTimeStampTester

Used to measure timestamp accuracy by sending a large number of messages to one channel to the other and checking their timestamps. The setup includes an example of how to use time domain handling.

Simple console application

Sets up two channels and sends messages back and forth between them.

Thread

Runs several threads that send and receive messages on different channels.

Tx

A traffic generator. It opens a channel, and sends bursts of messages to the channel for a set period of time.
