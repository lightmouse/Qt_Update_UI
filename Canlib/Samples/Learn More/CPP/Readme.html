C++ samples

These are some samples written in C++.

candemo

This program lists channel data and can transmit and receive messages on all channels.

CPPClassTest

This project contains a wrapper class called canlib which has a channel handle as an instance variable, to support a more object oriented programming style. It has a test program which uses this class to send some messages on a channel.

Notify

This program has a simple GUI written in MFC. It uses canSetNotify to listen for messages. It can also send messages to the channel.

qtcandemo

A simple program with a GUI written in QT.
