.NET Samples

This is a collection of sample programs. They are tested with Visual Studio 2010 and 2012. Some of them might also work with older versions. There are a few samples made in Visual Studio 2005 as well.

C# programs

Events

A simple example of events.

Remote device configuration

Demonstrates how to read and write configurations for a remote device using Kvrlib.

Remote device discovery

Uses Kvrlib to find remote devices.

Remote connection

Connects to a remote device and prints incoming messages.

Automatic responses

A program which uses object buffers to automatically respond to incoming messages.

Periodic transmissions

This example shows three ways of transmitting messages periodically.

Databases

A program which demonstrates various function in Kvadblib.

Notifications

A demonstration of asynchronous notifications in Canlib.

Logger

A program which saves incoming messages as a CSV file.

DBLog

A variation of the above, but uses a database to interpret the messages into signal values and saves those.

C++ programs

Events

Shows how to use event handles in a C++ application.

Notifications

Demonstrates how to use asynchronous notifications, similar to the C# sample.

Visual Basic programs

Introduction

This program demonstrates some of the most basic Canlib functions, such as opening a channel and writing and receiving messages.

Events

Demonstrates how to use event handles, virtually identical to the C++ sample.

Automatic transmissions

An example of how to use object buffers for periodic transmissions.
