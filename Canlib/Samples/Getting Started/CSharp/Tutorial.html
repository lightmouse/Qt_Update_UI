Canlib C# tutorial

Welcome!

This tutorial is intended to help programmers who are new to programming with Canlib. In this series, we show some of Canlib's most important functions and teach you how to use them.

These projects are made in Visual Studio 2010. We have also made Visual Studio 2008 projects for the first three parts. Parts 4-8 involve components which require .NET version 4 or higher.

Part 0: Preliminaries

Some useful things to know before you start programming.

Part 1: Hello world!

This first part will teach you how to open a channel, write a message to the channel, and close the channel.

Part 2: Receiving messages

In the second part, we show you how to wait for messages and read them.

Part 3: CanIoCtl

The third part demonstrates the various functionalities of the canIoCtl call.

Part 4: A GUI application

Here we demonstrate how you can create a graphical application which writes and sends messages.

Part 5: Events

This is a variation of the program part 4. It shows how to listen for messages on a channel by using event handles.

Part 6: Databases

This sample demonstrates how you can use CAN databases in your applications.

Part 7: More databases

A program which receives information, interprets it using a database and puts it into a chart.

Part 8: Sending messages with databases

This sample shows how you can take real world data, convert it into a CAN message and send it to the channel.
