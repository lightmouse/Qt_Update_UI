Canlib Python tutorial

Welcome!

This tutorial is intended to demonstrate how to use some of the most important parts of Canlib.

The Canlib wrapper

In order to make calls to Canlib, we have provided a wrapper in canlib.py. The most important parts of this wrapper is the canlib class, which initializes the library and wraps the calls to the dll file, and the canChannel class, which represents a channel handle and wraps most calls which take a channel handle as a parameter.

Wrappers for other libraries in the SDK are also available.
