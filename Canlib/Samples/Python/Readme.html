Python samples

This section contains wrappers for using the various libraries in Python. Most of them also contain some tests to show how to use them.

canlib

Wraps calls to Canlib. It contains one class whose constructor wraps the dll and that has methods for different Canlib calls. It also has one class (canChannel) which represents a channel handle, to support a more object oriented programming style. Also has a main method which tests the class by opening a channel and printing any incoming messages.

can_echo

This program takes two channel numbers as parameters and opens those two channels, calling one the server an one the client. It then starts two threads which run concurrently: the client writes messages on its channel and waits for the server to reply, while the server waits for messages and replies to anything it receives, with the identifier increased by 100.

kvaMemoLibXml

Wraps calls to kvamemolibxml.dll

kvDevice

The kvDevice class contains many useful methods for Canlib and Kvmlib calls. Also conatins some tests in the main method.

kvMemoConfig

Wraps the kvMemoConfig.dll functions.

kvmLib

Contains methods for calling kvmLib, as well as some examples of how to use them.

kvrLib

Wraps calls to Kvrlib. Includes a class for discovery handles. Some tests are included in the main method.
