Other samples

This section contains sample code for various APIs which didn't fit into any of the other sections. Most of the programs are written in C or C++.

j1587

This program demonstrates how to work with j1587 units.

j1699

Demonstrates how to work with j1699. This is a modified version of a test program published here originally.

joystick

A C program which reads the position of a joystick and uses a database to translate the movement into signals.

LIN

A tutorial and a couple of other samples for LIN.

REST

A couple of examples (one web application and one Android app) of how to use the REST API.

rp1210

Two sample .ini files for RP1210.

Sing

A sample program for working with the Softing emulation layer.

SingleWire

Shows how to work with SWC.

Vector

Shows how to work with the Vector API.
